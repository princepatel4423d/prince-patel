## Getting Started

### Prerequisites
- Node.js & npm

### Install
Clone the repo and install dependencies:
```bash
git clone https://github.com/princepatel4423d/prince-patel.git

cd server
npm install

cd webapp
npm install
```

### Run Locally
Start development server:
```bash
npm run dev
```
Open http://localhost:3000 (or the port shown in your terminal).
